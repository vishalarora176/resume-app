Key Projects

Overview Page

Dev Stack: SAPUI5, Annotation and Templates

Tools: Maven, GIT, JAWS, JIRA

Overview Page is a Smart Template library that is a part of Fiori elements and is developed by using SAPUI5. Customers can make use of the library to develop their custom applications within a couple of days that provide the experience of multiple Fiori applications in one Dashboard. The dashboard consists of various cards and each card represent a different Fiori application that is connected to the application via individual oData services.

Presales Readiness – Phase II

Dev Stack: SAPUI5 - Mobile, JavaScript, jQuery, HANA

The Presales Readiness is the renovation of the old Presales Readiness Council Tool and it is built on the SAPUI5 Mobile platform. It is fully responsive and accessible across all the screen types (Desktop/Tablet/Phone). The data is stored in HANA and is powered by oData Services to make it real time and fast. The tool has FIORI inspired UI and customizable Dashboard and Advanced search functionality.

Demo Registration

Dev Stack: HTML5, CSS3, JavaScript, jQuery, React JS, Redux

Demo Registration is a tool that helps the Presales People to register their Demos for a prospective client. This involves a series of steps like registration, request and approval of demos. The UI is entirely component based and uses React to power the rendering of the components in the application. The project runs on complex logics - both in front and back end.

Presales Central

Dev Stack: HTML5, CSS3, JavaScript, jQuery, Bootstrap

The Presales Central is and centralized tool to access and have the data flown from multiple tools to one central location. This tool manages the basic functionality of multiple tools allowing the user to access different forms and tools in one place. It is built on HTML/CSS using the scripting languages like JavaScript and jQuery and various plugins. The tool is compatible with desktop and tablets. Presales central makes use of various UI plugins and is developed with in MVC architecture following pure OOPS concepts.
